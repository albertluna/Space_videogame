instructions:
	Pulsa el botón espacio para empezar a jugar.
	Para moverte por el juego, puedes hacerlo con WASD o con las flechas.
	Cuando te salga la pantalla de game over o victoria, pulsa el botón R para volver a empezar.

explanation:
	El juego va de una nave kamikaze que tiene que colisionar con la nave central para ganar, a su alrededor hay 3 naves protectoras. El jugador tiene que evitar chocar con las naves protectoras y finalment chocar con la nave central, así ganará el juego. 
	Si el jugador intenta huir de la zona de combate, se le redigirá automáticamente a la zona de combate. No puede huir.
